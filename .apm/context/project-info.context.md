## RepoSweep v2

GitHub repo management dashboard — toggle visibility, archive, rename, delete repos from one place.

### Features

- Repo management dashboard
- Commit analytics
- Bulk repo operations (visibility, archive, rename, delete)

### Tech Stack

- Astro 5 SSR on Cloudflare Workers
- Vue 3 interactive islands
- Tailwind CSS 4
- Cloudflare KV for sessions
- GitHub OAuth + Octokit
- TypeScript strict
- pnpm
