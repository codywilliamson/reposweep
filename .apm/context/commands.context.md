## Commands

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start local dev server |
| `pnpm build` | Build for production (Astro build + assets ignore) |
| `pnpm preview` | Preview production build locally |
| `pnpm deploy` | Build and deploy to Cloudflare Workers |
| `pnpm lint` | Run ESLint |
| `pnpm typecheck` | Run `astro check` for type checking |
| `wrangler secret put <KEY>` | Set production secrets on Cloudflare |
